# Description

	This repository contains my homework project for the language processing course.
	Its objective is to try to extract all possible information of window titles for its use with a daemon *1 that detects what files and programs are in the screen focus.
	*1  The daemon was part of a work application and is not included.

# Contents

	Readme.txt			General repository information
	Makefile			Just a Make file
	Window_Titles.l	Contains code for the lexical analyser [Flex]
	Window_Titles.y	Contains code for the syntactic analyser [Byson]
	Window_Titles.txt	Contains test window titles
